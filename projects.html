Introduction to Data Analysis using Microsoft Excel

In project1, I learnt the foundation of data analysis with Microsoft Excel using sales data from a sample company. I learnt how to use sorting and filtering tools to reorganize data and access specific information about the data. I also learn about the use of functions like IF and VLOOKUP functions to create new data and relate data from different tables. Finally, I learnt how to create PivotTables to summarize and look at comparisons within the data. These skills will help me efficiently perform data analysis on many different types of data and give me the foundation to expand my toolbox as I explore other strategies for data analysis.
